一个现代化的 Discord 机器人开发和管理平台。

## ✨ 特性

- 🤓 **一键部署** - 快速启动和停止 Discord 机器人
- 🤓 **实时监控** - 实时日志查看和状态监控
- 🤓 **项目管理** - 创建、编辑、删除多个机器人项目
- 🤓 **响应式设计** - 完美支持桌面端和移动端

## 🚀 安装部署

### 环境要求

- Node.js 18+
- pnpm 8+

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/cmbh0/nextdiscordbot.git
cd nextdiscordbot

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

访问 http://localhost:3000 开始使用！

### 生产部署

```bash
# 构建项目
pnpm build

# 启动生产服务器
pnpm start
```

## 📁 项目结构

```
nextdiscordbot/
├── app/                    # Next.js App Router
├── components/            # React 组件
├── lib/                   # 工具函数和类型定义
└── public/                # 静态资源
```

## 🔧 技术栈

- **Next.js 16.2.0** - React 框架
- **React 19.2.4** - UI 库
- **TypeScript 5.7.3** - 类型系统
